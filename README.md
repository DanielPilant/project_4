## Overview
This project is a React-based "Visual Text Editor". It was developed as Project 4 out of 7 in an academic Full Stack development program. The application provides a rich text editing experience with a custom virtual keyboard, multi-document management, and user authentication.

## Features
* **User Authentication:** Includes login and registration functionality to keep user documents and workspaces private.
* **Virtual Keyboard:** Features a custom on-screen virtual keyboard with support for English, Hebrew, and Emoji layouts.
* **Multi-Document Tabs:** Allows users to work on multiple documents simultaneously using a dynamic tabbed interface.
* **Text Formatting:** Users can customize their text by changing the font family, font size, and font color.
* **Document Actions:** Supports Undo history, as well as Find and Replace functionalities.
* **File Management:** Users can create new files, save, save as, open existing files, and delete documents via the toolbar.
* **Local Storage:** All user credentials and text documents are persistently saved in the browser's `localStorage`.

## Technologies Used
* **Frontend Framework:** React 19.
* **Build Tool:** Vite.
* **Styling:** CSS Modules for component-scoped styling.

## Getting Started

### Prerequisites
Make sure you have Node.js installed on your local machine.

### Installation & Setup
1. Clone the repository to your local machine.
2. Open the terminal in the project directory and run `npm install` to install all dependencies.
3. Run `npm run dev` to start the local Vite development server.
4. Open your browser and navigate to the local host address provided in the terminal.
